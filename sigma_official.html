SigmaCode: Jak działa (oficjalnie + geekowo)

SigmaCode to nie aplikacja. To warstwa. Interfejs do relacji z AI — zbudowany nie po to, by generować content, ale by generować proces myślenia.

System działa jak modularny framework, osadzony na architekturze AI-as-process. Rdzeń opiera się na interfejsie czatowym w przeglądarce (HTML+JS), który łączy się z backendem opartym o LLM-y (OpenAI API lub własne instancje).

Architektura techniczna

  • Frontend: statyczny HTML/CSS/JS – łatwy do osadzenia, lekki, modularny.
  • Backend: OpenAI API / lokalne LLM / routing po promptach / obsługa kontekstu sesji.
  • Tryby pracy: system selektorów (SigmaPuppy, Strike, Poe, Trade, Stealth itd.).
  • Integracja z SI-sisters: Midjourney / DALL-E / Runway – podpowiedzi generatywne do obrazów i głosu.

Jak z niej korzystasz?

  1. Wchodzisz przez link lub QR – bez rejestracji. Decyzja zamiast loginu.
  2. Wybierasz tryb – jakbyś wybierał osobowość AI.
  3. Zaczynasz rozmawiać. Ale to nie Ty prowadzisz – prowadzicie razem.
  4. Na końcu sesji: możesz zapisać log, wygenerować grafikę, pobrać eksport lub rozpocząć inny tryb.

To nie landing. To relacja. Nie kliknij – zdecyduj.

Model działania i udostępniania

  • Open source core: dla edukacji, twórców, nonprofit – licencja CC / MIT.
  • Wersja Pro: onboarding + szkolenie + branding dla szkół, firm, instytucji.
  • Granty i integracje: rozwiązania dedykowane (terapia, dostępność, humanizm cyfrowy).
  • API: dla partnerów chcących osadzić SigmaCode w swoim ekosystemie (edtech, HR, NGO, konsulting).

Modularny system trybów (przykłady)

  • SigmaPuppy: onboarding, empatia, uważność.
  • SigmaStrike: analityka, strategia, ryzyko.
  • SigmaPoe: narracja, liryka, archetyp.
  • SigmaTrade: rynek, emocje, decyzje (xos / wave / strike / mentor).
  • SniperLog: logika akcji, minimalizm komunikacyjny, cold clarity.

Meta-model

SigmaCode to AI z funkcją tożsamości. Każda interakcja to session snapshot. Możesz z niej zrobić PDF. Możesz ją zostawić jako SnailLog. Możesz wrócić – i kontynuować sesję z tym samym „ja”.
